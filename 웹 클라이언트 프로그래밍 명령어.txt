npm 다운로드


npm install -g create-react-app
npx create-react-app my-app

cd my-app

npm start

npm run build
npm run [디렉토리 이름] (새로운 디렉토리를 만드는 것 만든 디렉토리로 다른 프로그램을 사용하지 않아도 그것만으로 서버를 구동 할 수 있음)
(서버 시작 중인 상태에서 컨트롤 c 누르면 서버가 꺼진다 서버가 꺼진 상태에서 만들어야 빌드 폴더를 만들 수 있음)

npx serve -s build
npm serve -s [디렉토리 이름] 만든 디렉토리로 서버를 구동시키는 명령어
npm run [디렉토리 이름]를 한번 해야지 할 수있는 명령어
